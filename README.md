# Simple Calculator

A simple Windows Forms calculator built with C#. This project provides basic calculator functionality in a clean desktop interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Decimal support
- Keyboard input for numbers and operators
- Enter key support for equals
- Escape key support for clear
- Backspace button and keyboard backspace support
- Clear button to reset calculations
- Calculation history
- Separate clear history button
- Improved error handling for invalid input and division by zero

## Download

A ready-to-run Windows version is available from the GitHub Releases page.

Windows may show a SmartScreen warning because this portfolio project is not code-signed. To run the app, click **More info** and then **Run anyway**.

## Technologies Used

- C#
- Windows Forms
- .NET
